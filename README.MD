Sobre O Projeto:
Esse projeto do Hospital Das Clinica tem como base ajudar pessoas com idade mais avançada, a marcar consultas com mais facilidade e também
responder algumas duvidas que eles possam ter ao realizar o agendamento da consulta. Esse Projeto sendo a continuidade do site anterior a esse, contendo mais features como agendar consultas e um layout mais agradável, dessa vez usando React e Typescript
